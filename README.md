# 📚 Library Management System

## 🚀 Overview

This **Library Management System** is built using **JavaFX, ORM, Java, and MySQL**, covering fundamental Java concepts while implementing a real-world application.

## ✨ Features

- **Book Borrowing System** 📖 - Users can borrow and return books efficiently.
- **User & Admin Panels** 🏛️ - Separate interfaces for users and administrators.
- **Branch & Book Management** 📂 - Manage multiple library branches and their book inventory.
- **Transaction View Section** 📊 - Track book lending and return transactions.

## 🛠️ Technologies Used

- **JavaFX** - For the UI and frontend.
- **Java** - Core logic and backend development.
- **ORM (Object-Relational Mapping)** - For database interactions.
- **MySQL** - Database management for storing records.

## 📌 Installation & Setup

1. **Clone the Repository**

   ```bash
   git clone https://github.com/your-repo/library-management.git
   cd library-management
   ```

2. **Setup MySQL Database**

   - Create a MySQL database and import the provided SQL schema.
   - Update database credentials in the configuration file.

3. **Run the Application**

   - Open the project in an IDE (like IntelliJ IDEA or Eclipse).
   - Compile and execute the main class.
